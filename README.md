This repository contains Java programs developed during a one-week placement training program, focusing on strengthening core programming concepts and preparing for technical interviews. The exercises cover fundamental topics such as arrays and strings, along with practical applications of sets and the HashSet class. Through these programs, I practiced efficient data handling, manipulation techniques, and problem-solving approaches that are commonly encountered in interview scenarios. Each program is written with clarity and includes comments to aid understanding, serving both as a record of my learning and a reference for future practice.
